
# Airline Management System

A Java-based airline management system developed as part of a university course project.

## Features
- Authentication and role-based authorization (admin and employee)
- Flight creation and management operations
- Ticket purchasing functionality
- File input/output operations for data persistence
- Multithreaded components for concurrent operations
- Java Swing based graphical user interface
- Object-oriented backend design

## Technologies
- Java  
- Java Swing  
- Object-Oriented Programming (OOP)  
- Multithreading  
- File I/O  
- Authentication & Authorization

## Project Structure
- Modular backend class design
- Java Swing based GUI layer

## Team
- Developed by a 2-person team

## Notes
This project emphasizes OOP, authentication, file handling, multithreaded programming, and GUI development in Java.
